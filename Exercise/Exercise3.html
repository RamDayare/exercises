1. position:static

The default positioning for all elements is position:static, which means the element is not positioned and occurs where it normally would in the document. Normally you wouldn't specify this unless you needed to override a positioning that had been previously set.Elements renders in order, as they appear in the document flow. This is default. If we are adding attributes like "left:10px;top:100px;",would not effect the current position of Image.And also the description text is in main document i.e. in main DIV element.
2. position:relative

If you specify position:relative, then you can use top or bottom, and left or right to move the element relative to where it would normally occur in the document. The element is positioned relative to its normal position, so "left:60" adds 60 pixels to the element's LEFT position. As we are using relative position so the right side description moves out of the document i.e. Main DIV element.
