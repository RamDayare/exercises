3. position:absolute

When you specify position:absolute, the element is removed from the document and placed exactly where you tell it to go. The element is positioned relative to its first positioned (not static) ancestor element.So we can see Image is positioned out of the document by adding attribute like "left:-20px;".
4. position:fixed

The element is positioned relative to the browser window.If you resize ,scroll up and scroll down the window then image location would not change when position is fixed.
